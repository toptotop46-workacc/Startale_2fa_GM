# Startale 2FA и GM

Один проект для двух квестов Startale (Soneium): **2FA (passkey)** и **GM (Send GM back)**.

- Запуск **по умолчанию** (`python main.py`): для **каждого** кошелька из `keys.txt` создаётся **один** браузер (AdsPower), в нём выполняется **2FA**, и **сразу после этого** выполняется **GM**, если он ещё нужен.
- Опционально: только 2FA (`python main.py --2fa-only`) или только GM без passkey (`python main.py --gm-only`).
- Перед **запуском браузера** для каждого кошелька проверяется: уже сделан 2FA и GM сейчас не нужен (5/5 выполнено или время следующего GM ещё не наступило)? Если да — браузер **не открывается**, кошелёк пропускается.

## Как это работает (простыми словами)

1. **Вы запускаете программу** — она читает список кошельков из `keys.txt`.
2. **Для каждого кошелька по очереди:**
   - Сначала программа **проверяет**, нужно ли что-то делать: уже настроен 2FA? Уже сделаны все 5 GM? Если для этого кошелька всё готово — он **пропускается**, браузер не открывается.
   - Если что-то нужно — открывается **один браузер AdsPower** для этого кошелька.
   - В браузере выполняется **2FA (passkey)** — вход через кошелёк и настройка двухфакторной аутентификации через расширение Bitwarden (Привязка -> Квест засчитывается -> Отвязка)
   - Затем в том же браузере, если нужно, выполняется **GM** — отправка приветствия GM столько раз, сколько ещё осталось до 5/5.
3. **Режимы запуска:**
   - `python main.py` — по умолчанию делаются и 2FA, и GM (в одном браузере на кошелёк).
   - `python main.py --2fa-only` — только настройка 2FA, без GM.
   - `python main.py --gm-only` — только отправка GM (2FA не трогаем).
4. **Результаты** сохраняются в `quest_results.json` — программа запоминает, когда следующий GM можно отправить, и не открывает браузер лишний раз.

Итого: один кошелёк = один браузер за запуск; лишние кошельки (где уже всё сделано) пропускаются автоматически.

## Требования

- Python 3.11+
- [AdsPower](https://www.adspower.net/) (с запущенным локальным API)
- API ключ AdsPower

## Установка

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Установите браузер для Playwright:

```bash
playwright install chromium
```

## Настройка

### 1) Приватные ключи

В файле `keys.txt` в корне проекта — по одному ключу на строку (hex 64 символа, с `0x` или без). Используются для обоих квестов (2FA и GM).

### 2) API ключ AdsPower

В файле `adspower_api_key.txt` — ваш API ключ AdsPower.

### 3) Прокси

В файле `proxy.txt` можно указать прокси для запросов к порталу и mail.tm. Формат: `host:port` или `host:port:user:pass`.

## Запуск

python main.py
